# Porsait

## English

Porsait is a minimal, offline-first, open source note-taking web application.
It focuses on speed, simplicity, and organizing thoughts into small, connected notes.

### Why Porsait?
Most note-taking apps are either bloated or too limited.
Porsait keeps things fast and simple, without accounts, tracking, or cloud sync.

### Features
- Fast note creation
- Tag-based organization
- Offline storage using localStorage
- Dark theme
- No accounts, no tracking, no cloud

### Tech Stack
- HTML
- CSS
- Vanilla JavaScript

### How It Works
All notes are stored locally in your browser.
Porsait does not collect or send any data.

### Demo

### Status
Early development. Core features are being built.
